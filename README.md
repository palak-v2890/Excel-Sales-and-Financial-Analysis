# 📊 AtliQ Hardware: Sales & Finance Analytics

## 📌 Project Overview
This project demonstrates how data analytics can drive informed business decisions in both sales and finance at **AtliQ Hardware**.  

By transforming raw data into actionable insights, the project focuses on uncovering trends, monitoring performance, and supporting strategic planning using Excel’s business intelligence capabilities.

---

## ⚙️ Approach & Workflow

### 1️⃣ Data Preparation (ETL Process)
**Tool Used:** Power Query (Excel)

- Combined and transformed data from multiple sources  
- Standardized columns and handled missing values  
- Optimized data types for performance  

**Result:** Clean, structured datasets ready for analysis  

---

### 2️⃣ Data Modeling  
**Tool Used:** Power Pivot (Excel)

- Established relationships between multiple tables  
- Enabled multi-dimensional analysis (time, region, category)  

**Result:** Scalable data model supporting interactive reporting  

---

### 3️⃣ Sales Insights  

**Key Areas:**
- **Customer Segmentation:** Identified top-performing customers and purchasing patterns  
- **Market Breakdown:** Analyzed trends across regions and time periods  

**Result:** Actionable insights to refine sales strategies and drive revenue growth  

---

### 4️⃣ Financial Insights  

**Key Areas:**
- **P&L Analysis:** Evaluated revenue streams and operational costs  
- **Profitability Drivers:** Identified factors impacting overall profitability  

**Result:** Improved budget alignment and stronger financial decision-making  

---

### 5️⃣ Reporting & Visualization  
**Tools Used:** Excel (Pivot Tables & Charts)

- Built clean, intuitive dashboards  
- Designed reports tailored for stakeholder usability  

**Result:** Clear and impactful data storytelling for business users  

---

## 🚀 Highlights & Learnings

- ✅ Mastered data preparation using Power Query  
- ✅ Built robust data models for deeper analysis  
- ✅ Implemented time intelligence using custom date tables  
- ✅ Designed user-friendly dashboards for decision-making  
- ✅ Identified key sales growth opportunities  
- ✅ Developed practical understanding of P&L analysis  

---

## 🛠️ Tools & Technologies

- Power Query (Microsoft Excel)  
- Power Pivot  
- Excel Pivot Tables & Charts  
- Excel 365  

---

## 📈 Conclusion

This project highlights the power of Excel as a business intelligence tool—showcasing how structured data, thoughtful modeling, and clear visualization can translate into meaningful business insights in both sales and finance.
